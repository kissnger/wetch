为什么需要 wetch

公众号编辑器不支持 flex、gradient、rgba、class 选择器。每次排版都要手写 table + 内联 style，重复枯燥。

wetch 的解法：写 Markdown，一键生成可直接粘贴到公众号编辑器的 HTML。5 种主题任选，经常用到的都帮你配好了。

特性

✅ 公众号兼容

纯 table + 内联 style 输出，flex/gradient/rgba/class 全部规避

🎨 5 种主题

极简红 · 科技蓝 · 自然绿 · 典雅紫 · 极简灰

🖼 自动封面

--cover 生成封面 HTML，Playwright 截图即用

📝 智能解析

标题、副标题、引用、列表、分隔线、加粗、代码高亮

🐍 零依赖

纯 Python 3，无需 Node.js、npm、数据库

⚡ 一键安装

pip install wetch，或者直接 clone 仓库用 PYTHONPATH

快速上手

# 安装
pip install wetch

# 转换文章
wetch article.md -o output.html

# 指定主题
wetch article.md --theme blue -o output.html
