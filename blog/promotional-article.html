公众号排版太痛苦？我写了个工具，用 Markdown 一键转兼容 HTML

发表于 2026.05.30 · #工具 #公众号 #开源

如果你做过公众号，一定经历过以下场景：

写好一篇稿子，复制到公众号编辑器里——
标题字号要调、引用要加底色、列表要对齐、代码块要自带宽高……
折腾半小时，终于排好了，一预览，手机端全乱了。

公众号编辑器不支持 flex、gradient、rgba、class 选择器。排版只能靠 table + 内联 style 手写。每个常用的样式的都要重新配一遍色号、边距、字号。

作为一个程序员，我忍了这件事很久——直到我决定写一个工具来解决它。

wetch：Markdown → 公众号 HTML

wetch 是一个纯 Python 的命令行工具。你写 Markdown，它帮你生成直接可用的公众号 HTML。

# 安装
pip install wetch

# 转换
wetch article.md -o output.html

就这么简单。output.html 可以直接打开复制，粘贴到公众号编辑器。

支持什么功能？

标准 Markdown 语法都支持：

标题（##）、副标题（###）、小节（####）、加粗（**）、引用（>）、列表（-）、代码块（```）、分隔线（---）。

额外支持：

日期、标签、作者元信息；封面文字自定义；5 种主题配色。

5 种主题配色

极简红
default
科技蓝
blue
自然绿
green

purple
极简灰
minimal

切换主题只需要加一个参数：

wetch article.md --theme blue -o output.html

封面图也帮你搞定

公众号文章需要封面图。wetch 的 --cover 参数可以生成封面 HTML：

# 生成封面 HTML
wetch article.md --cover -o cover.html

# 用 Playwright 截图为 PNG
npx playwright screenshot \
  --viewport-size=1200,675 cover.html cover.png

封面风格匹配你选的主题——深色渐变背景 + 网格底纹 + 装饰圈 + 标题和副标题布局。专业感在线，不用再找设计师了。

实际的输入输出对比

Markdown 输入（demo-article.md）：

## 你的 AI Agent 正在替你花钱

### Robinhood 和 Meow 这周都在做同一件事

date: 2026.05.30
tags: AI安全, AgenticFinance
author: wetch 用户

我问你一个问题。

如果你的 AI Agent 被黑客控制了，它能花掉你多少钱？

**44% 的电商企业已经在接入 Agentic Commerce**

> "AI Agent 正在创造企业安全史上最大的攻击面。"

输出的 HTML 直接打开，就能看到格式完整的公众号文章。包含题头卡片、引用块、强调块、标签、底部来源——所有公众号编辑器做起来费劲的东西，都有了。

和其他方案比有什么优势？

在线编辑器（如 135、秀米）：功能强但要手动操作，没法走自动化流程。模板多了反而选择困难。

Markdown → 微信的在线工具：大多需要上传到第三方平台，有隐私风险。样式选项单一。

wetch：纯本地运行，零隐私泄露。可脚本化、可 CI/CD、可团队共享标准模板。开源，MIT 许可。

谁适合用

✅ 技术团队的技术博客运营——码农写 Markdown 就是舒适区
✅ 个人号主想节约排版时间
✅ 需要统一多篇文章排版风格的内容团队
✅ 想把自动化流程跑起来的内容中台
